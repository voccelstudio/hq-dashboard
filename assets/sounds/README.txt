AMBIENT SOUND FILES
====================
Place .mp3 (or .wav, .ogg) files here to replace the procedural audio.

The app looks for: assets/sounds/{id}.mp3
If the file is missing, it falls back to procedural generation.

Expected files (CC0/royalty-free sources below):
  rain.mp3    - Rain ambience loop
  wind.mp3    - Wind ambience loop
  forest.mp3  - Forest / birds ambience loop
  ocean.mp3   - Ocean waves ambience loop
  fire.mp3    - Fire crackling loop
  brown.mp3   - Brown noise / deep rumble loop

FREE DOWNLOAD SOURCES (all CC0 / royalty-free):
  https://mixkit.co/free-sound-effects/ambience/
